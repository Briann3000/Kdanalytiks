# Survey Builder

Our drag-and-drop builder makes it simple to construct clean, interactive questionnaires.

## Key Features

- **Flexible Question Types**: Select from standard text input, paragraph text, multiple choice (radio/checkboxes), drop-downs, linear rating scales, and file uploads.
- **Question Library**: Save commonly used questions to your personal library so you can reuse them across surveys.
- **Templates**: Choose from premade templates (marketing, feedback, political, academic) to skip manual setup.

## Designing a Survey

1. From the dashboard, click **New Survey**.
2. Select a category (Product, Academic, Marketing, Political).
3. Use the sidebar toolbox to add new questions.
4. Click on any question card in the center canvas to customize its label, options, placeholder, or marked requirement.
5. Click **Deploy** to publish the survey and start collecting responses.
